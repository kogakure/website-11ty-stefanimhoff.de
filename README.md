# stefanimhoff.de

This is the source of my [personal website][si] build with [Eleventy], [PostCSS], and [Gulp].

## Installation

```sh
pnpm install
```

## Running the Website

```sh
pnpm start
```

For more tasks have a look into the `package.json`.
